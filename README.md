Развлекательное приложение Procrastinate Pro+. Несмотря на огромные вложения в рекламу, последние несколько месяцев компания терпит убытки. Задача — разобраться в причинах и помочь компании выйти в плюс.
Есть данные о пользователях, привлечённых с 1 мая по 27 октября 2019 года:
- лог сервера с данными об их посещениях,
- выгрузка их покупок за этот период,
- рекламные расходы.
Предстоит изучить:
- откуда приходят пользователи и какими устройствами они пользуются,
- сколько стоит привлечение пользователей из различных рекламных каналов;
- сколько денег приносит каждый клиент,
- когда расходы на привлечение клиента окупаются,
- какие факторы мешают привлечению клиентов.

План работы:

- загрузка данных о визитах, заказах и рекламных расходах из CSV-файлов в переменные;
- изучение данных и предобработка;
- исследовательский анализ данных, а именно составление профилей пользователей, определение минимальной и максимальной даты привлечения пользователей;
- построение таблицы, отражающей количество пользователей и долю платящих из каждой страны;
- построение таблицы, отражающей количество пользователей и долю платящих для каждого устройства;
- построение таблицы, отражающей количество пользователей и долю платящих для каждого канала привлечения;
- подсчёт общей сумму расходов на маркетинг;
- построение графика с визуализацией динамики изменения расходов во времени по неделям по каждому источнику;
- анализ окупаемости рекламы c помощью графиков LTV и ROI, а также графиков динамики LTV, CAC и ROI;
- проверка конверсии пользователей и динамику её изменения;
- выводы.
